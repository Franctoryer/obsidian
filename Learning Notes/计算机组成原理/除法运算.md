# 1 恢复余数法

每次加上除数的补码
- 余数为正，说明够减，上商1
- 余数为负，不够减，上商0，恢复余数

余数逻辑左移一位

# 2 加减交替法
设余数为 $R_i$，除数为 $y^*$

- 当 $R_i >0$，上商1，做 $2R_i-y^*$
- 当 $R_i < 0$，上商0，做 $2R_i+y*$

# 3 补码的加减交替法
第一步：
- 若 $[x]_{补}$ 和 $[y]_{补}$ 同号，计算 $[x]_{补}-[y]_{补}$
- 若 $[x]_{补}$ 和 $[y]_{补}$ 异号，计算 $[x]_{补}+[-y]_{补}$

|$[x]_{补}$ 与 $[y]_{补}$|商|新余数 $[R_{i+1}]_{补}$|
|:-:|:-:|:-:|
|同号|1|$[R_{i+1}]_{补}=2[R_i]_{补}+[-y]_{补}$|
|异号|0|$[R_{i+1}]_{补}=2[R_i]_{补}+[y]_{补}$|
