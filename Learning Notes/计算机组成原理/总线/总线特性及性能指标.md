# 1 总线特性
1. 机械特性：尺寸、形状、管脚数、排列顺序
2. 电气特性：传输方向和有效的电平范围
3. 功能特性：每根传输线的功能，详见[[总线的分类]]
4. 时间特性：信号之间的时序关系


# 2 总线的性能指标
1. 总线宽度：即数据线的根数
2. 标准传输率：每秒传输的最大字节数（MBps）
3. 时钟同步/异步：同步、不同步
4. 总线复用：地址线与数据线复用（为了减少芯片的管脚数）
5. 信号线数：地址线、数据线和控制线爱的总和
6. 总线控制方式：突发、自动、仲裁、逻辑、计数
7. 其他指标：负载能力
