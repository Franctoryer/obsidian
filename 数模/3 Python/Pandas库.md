# 1 读取文件
1. `read_csv(<filepath>, nrows=<num>,skiprows=<num>)` 读取csv文件
2. `read_excel()` 读取excel文件

# 2 数据结构
- `Series` ：一维
- `Dataframe` ：二维

# 3 常用函数
- `describe()` 查看各个字段基本的数字特征
- `value_sort` 排序
- `value_counts()` 计数
- `astype` 改变数据类型
- `sum` 、`mean` 、`var` ……


# 4 常用属性
- `index` 查看行字段
- `colmuns` 查看列字段
